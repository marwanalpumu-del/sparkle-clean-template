SparkleClean Documentation

Professional Cleaning Service Landing Page Template

1. Overview

SparkleClean is a modern, responsive landing page designed for cleaning companies. It features a smart cost calculator and seamless payment integration logic.

2. Folder Structure

The main files are located in the HTML/ folder:

  • index.html: The main structure of the website.
  • assets/css/style.css: Contains all the visual styling and layout rules.
  • assets/js/main.js: Contains the calculation engine and booking logic.

3. How to Change Prices

To update the service rates, open assets/js/main.js and locate the rates in the HTML select options or the JS logic. You can easily adjust the multipliers.

4. Payment Setup

The "Book Now" button is programmed to redirect to PayPal. To use your business account, edit the merchantEmail variable in assets/js/main.js.

5. Customization

Colors are managed via CSS Variables in style.css. Simply change the --primary color code to update the entire theme's look.
