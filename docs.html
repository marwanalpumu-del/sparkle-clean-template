Template Setup Guide | دليل إعداد القالب

English Guide

Welcome! Follow these steps to set up your template:

1. Configuration

Open assets/js/main.js and update your WhatsApp number:

whatsappNumber: "9665XXXXXXXX",
baseRate: 5 // Price per m²

2. Files Included

  • index.html (English Version)
  • index-ar.html (Arabic Version)
  • style.css (Central Styles)
Tip: Change --primary in style.css to update the whole brand color.

الدليل العربي

أهلاً بك! اتبع هذه الخطوات لتجهيز قالبك الخاص:

١. الإعدادات الأساسية

افتح ملف assets/js/main.js وضع رقم الواتساب الخاص بك:

whatsappNumber: "9665XXXXXXXX",
baseRate: 5 // سعر المتر المربع

٢. الملفات المرفقة

  • index.html (النسخة الإنجليزية)
  • index-ar.html (النسخة العربية RTL)
  • style.css (ملف التنسيق الشامل)
نصيحة: لتغيير لون الموقع بالكامل، قم بتغيير كود اللون في --primary داخل ملف CSS.
