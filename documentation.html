Documentation: SparkleClean Premium Template ​1. Introduction ​SparkleClean is a professional, high-performance HTML5 template designed for cleaning service providers. It features a modern, responsive design with advanced business logic, including a live cost estimator and WhatsApp booking integration. ​2. Main Features ​Multi-Language Support: Full LTR (English) and RTL (Arabic) versions included. ​Theme Switching: Built-in Dark & Light mode toggle with local storage persistence. ​Interactive Cost Estimator: Real-time pricing calculation based on area size. ​Conversion Optimized: Direct WhatsApp booking system for instant lead generation. ​Fully Responsive: Tested on multiple devices (iOS, Android, Windows, Mac). ​W3C Validated: Clean, error-free code optimized for SEO. ​3. Folder Structure ​Inside the main package, you will find: ​/HTML/index.html — The English version (LTR). ​/HTML/index-ar.html — The Arabic version (RTL). ​/HTML/assets/css/ — All stylesheets (style.css). ​/HTML/assets/js/ — Core JavaScript files (main.js). ​/HTML/assets/images/ — Optimized image assets. ​/Documentation/ — This documentation file. ​4. Installation & Customization ​A. How to Set Your WhatsApp Number: ​Open assets/js/main.js. ​Locate the variable whatsappNumber. ​Replace the placeholder with your phone number (including country code, e.g., 966500000000). ​B. How to Edit Pricing: ​In assets/js/main.js, find the calculation function. ​Update the pricePerMeter variable to match your service rates. ​C. Branding (Colors & Fonts): ​Colors: All colors are managed via CSS Variables. Open assets/css/style.css and edit the :root section. ​Fonts: We use Inter for English and Cairo for Arabic via Google Fonts. ​5. Important Note on Visual Content ​Before & After Section: The provided "Results" image features a Bathroom Deep Clean. ​If you wish to use your own images, ensure they are high-resolution and maintain the aspect ratio for the best visual experience. ​6.
