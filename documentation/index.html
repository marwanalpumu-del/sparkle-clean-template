SparkleClean Documentation

Version: 1.8.5 | Author: Maymona

1. Introduction

SparkleClean is a professional, high-performance HTML5 template designed for cleaning service providers. It features a modern, responsive design with advanced business logic, including a live cost estimator and WhatsApp booking integration.

2. Main Features

  • Multi-Language Support: Full LTR (English) and RTL (Arabic) versions included.
  • Theme Switching: Built-in Dark & Light mode toggle with local storage persistence.
  • Interactive Cost Estimator: Real-time pricing calculation based on area size.
  • Conversion Optimized: Direct WhatsApp booking system for instant lead generation.
  • Fully Responsive: Tested on iOS, Android, Windows, and Mac.
  • W3C Validated: Clean, error-free code optimized for SEO.

3. Folder Structure

/HTML/index.html        — English version (LTR)
/HTML/index-ar.html     — Arabic version (RTL)
/HTML/assets/css/       — Stylesheets (style.css)
/HTML/assets/js/        — JavaScript files (main.js)
/HTML/assets/images/    — Optimized images
/Documentation/         — This help file

4. Installation & Customization

A. WhatsApp Number

Open assets/js/main.js, locate whatsappNumber, and replace it with your number:

const whatsappNumber = "966500000000";

B. Pricing Logic

Update pricePerMeter in main.js to match your rates.

C. Branding

Colors: Edit the :root section in assets/css/style.css.

Fonts: Uses Google Fonts (Inter & Cairo).
