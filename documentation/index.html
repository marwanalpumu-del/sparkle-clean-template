SparkleClean Documentation

Version: 1.8.5 | Author: Maymona

1. Introduction

SparkleClean is a professional, high-performance HTML5 template. It features a modern design with central business logic for WhatsApp integration.

2. Main Features

  • Auto-Sync WhatsApp: Change the number in one place to update the whole site.
  • Multi-Language Support: Full LTR (English) and RTL (Arabic) versions.
  • Interactive Estimator: Real-time pricing calculation.

3. Installation & Customization

A. WhatsApp & Business Info

Open assets/js/main.js and update the COMPANY_SETTINGS object at the top:

const COMPANY_SETTINGS = {
    whatsappNumber: "966500000000", // Your WhatsApp number
    baseRate: 5                     // Your price per meter
};
Professional Hint: All buttons (Floating, Contact, and Booking) update automatically once you save the changes in main.js.
