SparkleClean Documentation

Version: 1.8.5 | Author: Maymona

1. Introduction

SparkleClean is a professional, high-performance HTML5 template. It features a modern design with central business logic for WhatsApp integration.

2. Main Features

  • Auto-Sync WhatsApp: Update your phone number once to change it across the entire site.
  • Multi-Language Support: Full LTR (English) and RTL (Arabic) versions included.
  • Interactive Estimator: Real-time pricing calculation based on area size.
  • Dark Mode Support: Built-in theme switcher with local storage persistence.

3. Installation & Customization

A. WhatsApp & Business Info

Open assets/js/main.js and update the COMPANY_SETTINGS object at the top of the file:

const COMPANY_SETTINGS = {
    whatsappNumber: "966500000000", // Replace with your international number
    baseRate: 5                     // Set your price per square meter
};
Professional Hint: All buttons (Floating Button, Contact Section, and Calculator) update automatically once you save changes in main.js. No HTML editing required!

B. Branding

Colors: Edit the :root variables in assets/css/style.css to match your brand colors.

Fonts: Uses Google Fonts (Inter and Cairo).

4. Folder Structure

/HTML/index.html        — English version (LTR)
/HTML/index-ar.html     — Arabic version (RTL)
/HTML/assets/css/       — Main Stylesheet
/HTML/assets/js/        — Core Logic (main.js)
/HTML/assets/images/    — Optimized image assets
/Documentation/         — This guide
